[sweet](https://github.com/duan602728596/sweet) 是一个webpack开发和生产环境的通用配置解决方案。

## 特点：

1. 支持React和Vue的开发，会根据配置自动添加所需要的loaders和plugins。  
2. 在开发环境下可以选择直接编译到指定目录（适用于Nwjs和Electron的开发），网站的开发可以使用基于Koa的服务器。
3. 在生产环境下可以编译后快速部署。
4. 支持react、vue的服务端的渲染。
